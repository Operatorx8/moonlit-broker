# Fatalis - 数值参数

## 基础属性

| 参数 | 值 | 说明 |
|------|-----|------|
| BASE_DAMAGE | 12 | 基础攻击伤害 |
| ATTACK_SPEED | 1.8 | 攻击速度（4.0 - 2.2） |
| DURABILITY | 287 | 耐久度（IRON × 1.15） |
| RARITY | UNCOMMON | 稀有度 |
| ENCHANTABILITY | 15 | 附魔系数（Katana 档） |
