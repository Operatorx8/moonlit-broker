# Changelog

## [Unreleased]

### Changed
- Nmap Host Discovery now grants Resistance V for 5 seconds (was 2 seconds).
- Updated Nmap tooltip/localization text in `zh_cn` and `en_us` to reflect the 5-second duration.

- Added particle feedback when Nmap Host Discovery shield activates.
- Added particle feedback when Nmap shield is interrupted by incoming damage.
- Added player-side particle feedback when detected hostiles increase Nmap penetration stacks.
- Added target-side particle feedback when Nmap penetration bonus damage is applied.
